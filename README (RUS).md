# Guardian of Health

Локальный, приватный AI-ассистент, который анализирует видеопоток с вашей веб-камеры в реальном времени для предотвращения цифровой усталости (напряжения глаз, сутулости) и выгорания без отправки вашего видео в облако.

## Возможности
- Анализ видео в реальном времени — использует вашу веб-камеру для отслеживания осанки.
- AI-оценка позы — MediaPipe определяет 33 ключевые точки тела.
- Расчёт угла осанки — измеряет сутулость в реальном времени.
- Умные уведомления — оповещает при сутулости (консоль + визуально).
- 100% приватность — вся обработка выполняется локально, данные не покидают ваш компьютер.
- Веб-интерфейс — просматривайте статистику осанки в любом браузере.
- Голосовые команды (опционально) — управляйте приложением без рук.
- Счётчик сутулостей — отслеживайте, сколько раз вы сутулились.

## Быстрый старт

### Требования
- Python 3.12+ ([Скачать](https://www.python.org/downloads/))
- Веб-камера (встроенная или внешняя)
- Git ([Скачать](https://git-scm.com/downloads))

### Установка.
# 1. Клонируйте репозиторий
git clone https://github.com/Anton-Sergeev-EA/Guardian_of_health.git
cd Guardian_of_health
# 2. Создайте и активируйте виртуальное окружение
python -m venv venv
source venv/bin/activate      # На Linux/macOS
venv\Scripts\activate         # На Windows
# 3. Установите зависимости
pip install -r requirements.txt
# 4. Запустите приложение
python guardian_final_working.py
# 5.Использование.
- Откройте браузер и перейдите по адресу http://localhost:5000
- Разрешите доступ к камере при запросе.
- Сядьте перед веб-камерой.
- Наблюдайте за углом осанки и счётчиком сутулостей в реальном времени.

# Архитектура
Веб-камера → OpenCV → MediaPipe Pose → Расчёт угла → Обнаружение сутулости → UI (Веб/Консоль)

# Ключевые компоненты:
- Захват видео: OpenCV (C++ движок для производительности).
- Оценка позы: MediaPipe (33 ключевые точки).
- Веб-сервер: Flask (MJPEG-стриминг + REST API).
- UI: HTML/CSS + JavaScript (автообновление статистики).

# Конфигурация:
